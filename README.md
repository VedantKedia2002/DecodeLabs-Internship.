🎓 Student Grade Calculator

📌 Overview
The Student Grade Calculator is a simple application that helps calculate a student's total marks, average percentage, and final grade based on input scores.

🚀 Features
- Input marks for multiple subjects
- Automatic total and average calculation
- Grade assignment based on percentage
- User-friendly interface (CLI/GUI)

🛠️ Technologies Used
- Language: (Java / Python / C++ / JavaScript)
- Tools: (VS Code / IntelliJ / Eclipse)


▶️ How to Run

For Java:
bash
javac Main.java
java Main

BASH:
python main.py

Enter marks: 80 90 70
